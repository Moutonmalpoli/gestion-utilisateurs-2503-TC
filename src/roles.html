Tableau des rôles

Nom Description
Invité Usager non identifié:
  • Créer un compte s’il dispose du code d’inscription
  • Demander à réinitialiser son mot de passe
Usager Utilisateur identifié:
  • Modifier les informations de son propre profil sauf son rôle
  • Parcourir la liste des utilisateurs enregistrés
Encadrant Usager avec privilèges accrus:
  • Consulter les profils individuels des usagers et encadrants
  • Modifier les informations des usagers (sauf leur rôle et leur mot de passe)
Administrateur Encadrant avec privilèges accrus :
  • Consulter le profil individuel de n’importe quel utilisateur
  • Modifier toutes les informations d’un utilisateur enregistré
  • Ajouter et supprimer un utilisateur
